# Formulaire

A simple and responsive form created using **HTML** and **CSS**.  
This project demonstrates basic form structure and styling without using JavaScript.

## Features

- User-friendly form layout
- Input fields with labels
- Submit button
- Clean and minimal design
- Responsive on different screen sizes
- Built with only HTML and CSS

## Technologies Used

- **HTML** – Structure of the form  
- **CSS** – Styling and layout
